# স্বাস্থ্য-সুরক্ষা (Shastho Surukha) – Unified Healthcare Management System UI/UX

## Description
স্বাস্থ্য-সুরক্ষা is a web-based healthcare platform UI/UX design that integrates essential services such as vaccination tracking, blood donation, doctor appointments, hospital bed reservations, and emergency services. Guests can access key features without registration. The design emphasizes **usability, real-time updates, and intuitive user experience**.
